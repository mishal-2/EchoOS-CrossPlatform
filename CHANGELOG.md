# Changelog

All notable changes to EchoOS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-12-02

### Added
- **Cross-platform support** for Windows and macOS
- **Voice biometric authentication** using Resemblyzer
- **Offline speech recognition** with Vosk
- **Modern PySide6 GUI** with intuitive interface
- **40+ voice commands** across multiple categories
- **Application discovery** system for installed apps
- **Command parser** with fuzzy matching
- **Text-to-speech feedback** using pyttsx3
- **Session management** with encryption
- **Accessibility features** for screen reading
- **Comprehensive documentation** and guides
- **Automated app discovery** in background
- **Manual command input** option
- **Activity logging** system
- **Status indicators** and real-time feedback

### Command Categories
- System control (shutdown, restart, sleep, lock)
- Application management (open, close, minimize, maximize)
- File operations (open, create, delete, list)
- Web operations (open website, search Google/YouTube)
- System information (battery, disk, memory, CPU, network)
- Volume control (up, down, mute)
- Accessibility (read screen, navigate, click, scroll)

### Security
- Voice biometric authentication
- Encrypted session storage
- 30-minute session timeout
- Secure user database
- Local-only processing (no cloud)

### Performance
- 92% authentication accuracy
- 9.85% Word Error Rate
- 93% command success rate
- 150ms average response latency
- 315MB memory footprint

### Documentation
- Comprehensive README
- Detailed installation guide
- Quick start guide
- Contributing guidelines
- API documentation
- Troubleshooting section

### Developer Tools
- Modular architecture
- Type hints throughout
- Comprehensive logging
- Error handling
- Unit test framework
- Code formatting tools
